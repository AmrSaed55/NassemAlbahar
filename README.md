\# 🌊 Nassem Albahar - Hotel \& Travel Booking Website



\## 📌 Overview



\*\*Nassem Albahar\*\* is a modern hotel and travel booking website designed to provide users with an easy, fast, and secure way to explore hotels, rooms, and travel tours.



The platform gathers the best hotel options in one place and allows users to book their stays with a smooth and user-friendly interface.



This project demonstrates front-end web development skills using modern animation and responsive design techniques.
